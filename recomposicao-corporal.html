O que é recomposição corporal?

Recomposição corporal é o processo de reduzir o percentual de gordura enquanto se preserva — ou até aumenta — a massa muscular. É diferente de simplesmente "emagrecer": o objetivo não é apenas perder peso na balança, mas mudar a composição do que esse peso representa.

O percentual de gordura corporal é a métrica mais relevante para avaliar a composição física. Duas pessoas com o mesmo peso podem ter aparências completamente diferentes dependendo de quanto desse peso é músculo e quanto é gordura.

Como a projeção é calculada

Esta calculadora usa a fórmula Mifflin-St Jeor para estimar o metabolismo basal e aplica o fator de atividade para calcular o TDEE. A partir disso, simula semana a semana o déficit calórico necessário para atingir o percentual de gordura alvo, projetando como o peso e a composição corporal vão evoluir ao longo do tempo.

Importantes: as projeções são estimativas. O corpo humano não é linear — há semanas de platô, retenção de água, variações hormonais. Use os números como referência e ajuste conforme os resultados reais a cada 3–4 semanas.
